Preguntas Frecuentes

Resolvemos tus dudas sobre el funcionamiento de la plataforma.

TecnoCorte cuenta con una agenda inteligente en tiempo real conectada a la base de datos de MySQL. En cuanto un cliente reserva un slot de horario, el sistema lo inhabilita instantáneamente para cualquier otro usuario.

¡Sí! Desde tu perfil de Cliente tendrás una sección de "Mis Citas" donde podrás cancelar o modificar tus reservas con límites de tiempo configurados por la administración.

Este proyecto de formación SENA está construido con Node.js y Express.js en el backend, motor de plantillas EJS y CSS para la experiencia visual, e interactividad mediante JS Nativo. Los datos se almacenan de forma segura en MySQL.
